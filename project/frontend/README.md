# AutoDocstring Frontend

React 18 + TypeScript + Tailwind CSS web UI for the Automated Python Docstring Generator.

## Setup

```bash
npm install
npm run dev
```

The dev server starts at **http://localhost:5173**.  
Make sure the backend is running at **http://localhost:8001** before using the UI.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Zustand (state management)
